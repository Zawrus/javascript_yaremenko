-Kiedy powstala


Jezyk JavaScript zostal opracowany w 1995 roku przez przedsiebiorstwo
Netscape Corporation oraz Sun Microsystem specjalnie w celu
uzycia go przy tworzeniu stron WWW.


-Dlaczego jest to jezyk skryptowy


JavaScript to obiektowy jezyk skryptowy, kiedy Java to obiektowy jezyk programowania.
Dodatkowo zarowno jeden, jak i drugi jezyk wymaga instalowania odpowiednich wtyczek.
Dlatego poczatki z JavaScript to takze troche wiedzy teoretycznej.


-Gdzie jest wykonywany JS (klient lub serwer)


JavaScript jest uzywany najczeciej przy tworzeniu stron www, zapewniajac ich
interaktywnosc oraz obsluga zdarzen, walidacja formularzy czy budowanie elementow
nawigacyjnych. Ale w tym jezyku mozna takze pisac pelnoprawne aplikacje
(korzystajac z frameworkow do budowania aplikacji)


-Dlaczego jest to bezpieczne narzedzie


JS, podobnie jak inne jezyki i srodowiska, jest bezpieczny, gdy projekty
w nim budowane sa traktowane z odpowiednia dbaloncia i zastosowaniem
najlepszych praktyk.


-Czy jest to jezyk obiektowy , jezli tak to dlaczego.


Java to obiektowy jezyk programowania, a JavaScript to obiektowy jezyk skryptowy.
Aplikacje w jezyku Java dzialaja w maszynie wirtualnej lub w przegladarce,
a kod JavaScript jest uruchamiany wylacznie w przegladarkach.

Element formularza ma rodzaj "button", nazywa sie przycisk i jego wartosc "Nowa Strona".
window.open("obraz.html","okienko","toolbar=no,directories=no,menubar=no,height=280,width=160,top=200,left=200");

Standardowa konstrukcja otwierajaca okno ma url "obraz.html",strona nazywa sie "okienko" i ma kilka ustawien:
-toolbar=no(ukrywa standardowy pasek narzedzi);
-directories=no(ukrywa przyciski katalogow);
-menubar=no(ukrywa menu przegladarki);
-height=280(wysokosc okna);
-width=160(szerokosc okna);
-top=200(polozenie okna wzgledem gory ekranu);
-left=200(polozenie okna wzgledem lewej strony ekranu).
