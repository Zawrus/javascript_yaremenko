    <h2>Teoria</h2>
   <h3>Rodzaje okienek:</h3> 
    <ol>
<li>Okienko alert – wyświetlająca okienko dialogowe z informacją i przyciskiem OK,</li> <br>
<li>Okienko prompt – generująca okno, w którym użytkownik może wpisać wartość, zwracaną
    następnie do programu.</li> <br>
<li>Okienko confirm służy do tego, aby użytkownik coś potwierdził (stąd nazwa confirm).</li>
    </ol>
<b>Przy nadawaniu nazw zmiennym należy przestrzegać pewnych zasad:</b> <br>
1. Nazwa zmiennej musi się zaczynać od litery, znaku "$" lub "_", może zawierać tylko litery (bez <br>
polskich znaków), cyfry oraz "$" i "_". <br>
<u>Poprawne nazwy zmiennych</u>: zmienna1, $wynik, _rozmiar, wynik1, Rekord, $_gracz <br>
<u>Błędne nazwy zmiennych</u>: 1zmienna, wynik gry, wysokosc&szerokosc, wynik.rekord, &gracz, <br>
wysokość, imie+nazwisko. <br>
2. Wielkość liter ma znaczenie. <br>
<u>Rekord i rekord</u> to dwie różne zmienne! Jest to bardzo częsta przyczyna błędów w kodzie <br>
początkujących programistów. <br>
3. Należy nie używać słów kluczowych JavaScript jako nazw zmiennych. <br>
<h3>Typy zmiennych w Javascript</h3> 
W JavaScript nie ma możliwości definiowania różnych typów zmiennych tak jak w C++, typy <br>
zmiennych są definiowane w trakcie ich przypisywania. <br>
Podstawowe zmienne liczbowe w JavaScript to: <br>
Number -> Typ liczbowy <br>
<ol>
 <li>zmienna całkowite</li> <br>
 <li>zmiennoprzecinkowe</li> <br>
</ol>
String -> Typ tekstowy <br>
Boolea -> Typ logiczny <br>
Null -> Pusta zawartość <br>
Undefined -> Typ niezidentyfikowany <br>
Object -> Obiekt <br>
<h3>Konwersji zmiennych.</h3>  
Wczytanie zmiennej z użyciem <b>prompt</b> wykonywane jest jako ciąg znaków dlatego konieczna jest <br>
konwersji z ciągu znaków na wartości liczbowe aby wykonywac obliczenia. <br>
<br>
<u>parseFloat(zmienna)</u> - konwersja zmiennej do typu float <br>
<u>parseInt(zmienna)</u> - konwersja zmiennej do typu int (liczbowego) <br>
<u>toString(zmienna)</u> - konwersja zmiennej do typu łańcuchowego <br>
